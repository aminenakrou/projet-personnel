# Churn Client – Pipeline ML complet

Ce projet fournit un pipeline de classification pour la prédiction du churn des clients, basé sur un ensemble d'algorithmes en stacking. Il expose une API REST sous Flask, orchestrée via Docker, et le suivi des modèles est intégré grâce à MLflow.

## Structure du projet
- data/
  - churn_sample.csv
- src/
  - train.py
  - predict.py
  - model_utils.py
  - api.py
- Dockerfile
- requirements.txt
- README.md
- .env

## Installation rapide
Voir le fichier README.md pour les instructions détaillées (Docker, MLflow, API).
